# Embedded Ultrasonic Radar

An embedded systems project demonstrating object detection and distance measurement using an Arduino Uno, HC-SR04 ultrasonic sensor, and servo motor. The project is being developed incrementally with proper documentation and version control.

## Objectives

- Measure object distance using ultrasonic sensing
- Rotate the sensor using a servo motor
- Build a simple radar scanner
- Learn embedded system design practices
- Maintain professional GitHub documentation

## Hardware Used

- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- SG90 Servo Motor
- Breadboard
- Jumper Wires
- USB Cable
